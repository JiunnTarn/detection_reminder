- /reminder：发送一次提醒

| 返回值        | 结果               |
| ------------- | ------------------ |
| Done          | 成功               |
| Error         | 发生错误           |
| Already Done. | 今天全部同学已完成 |



- /get_undone_student：获取今天还未完成的同学



- /exclude_by_name/：进入提交页面



- /exclude_by_name/index.html：get请求提交数据

| 传入值 | 类型   |
| ------ | ------ |
| name   | String |

| 返回值                 | 含义                    |
| ---------------------- | ----------------------- |
| xxx 提交成功。         | 成功                    |
| 找不到 xxx。           | 在学生数据表中找不到xxx |
| xxx 今天已经提交过了。 | xxx重复提交             |
